

## setup flow

next.js + shadcn/ui 프로젝트 생성  

```sh
pnpm dlx shadcn@latest create 
```

**prettier configurations** (in package.json)

```sh 
pnpm add -D prettier 
```

```json
{
  "prettier": {
    "singleQuote": true,
    "trailingComma": "all",
    "semi": false,
    "printWidth": 120
  }
} 
```
